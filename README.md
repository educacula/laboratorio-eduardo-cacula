# Laboratório Eduardo Caçula - GitHub Pages

Este ZIP já está no formato que você usa no GitHub Pages:

- index.html
- manifest.webmanifest
- sw.js
- assets/

## Como publicar

1. Descompacte o ZIP
2. Crie um repositório no GitHub
3. Envie todos os arquivos para a raiz do repositório
4. Vá em Settings > Pages
5. Em Source, escolha:
   - Deploy from a branch
   - Branch: main
   - Folder: /root
6. Aguarde o link do GitHub Pages

## Como instalar no celular

1. Abra o link do GitHub Pages no Chrome
2. Toque em:
   - menu ⋮
   - Adicionar à tela inicial

## Observação

Os dados ficam salvos no navegador do aparelho usando localStorage.
Se trocar de aparelho ou limpar os dados do navegador, será preciso repor os dados.
